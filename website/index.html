Objectives

The following are the objectives implemented in the ray tracer.

Primitives

Implemented the extra primitives cones, cylinders.

Refraction

This is acheived by casting a secondary ray using the index of refraction and Snell's Law. I then use Fresnel's law to combine the colours from the reflected and refracted ray.

Glossy Refraction

Instead of casting one refracted ray I cast multiple ray perturbed in random directions and average their colors.

Soft Shadows

To achieve soft shadows I replaced point lights by area lights. I implemented area lights as a light source with multiple bulbs in a square arrangement with a variable separation.

Texture Mapping

Texture mapping was implemented by generating a 2D normalized coordinates for each primitive which were then used to index into the texture image to retrieve a texture sample.

Bump Mapping

To acheive bump mapping I use the same way of generating 2D coordinates for a surface used for texture mapping. I then use the partial derivatives values of the bump map to perturb the normal of the surface.

Perlin Noise

I used the random noise generatred implemented by Ken Perlin to generate noise that I then used as input to a function that decided the intensity of the colour of the surface. For this example I used a function that generated a wood pattern, and the floor is generated using the raw generated noise.
